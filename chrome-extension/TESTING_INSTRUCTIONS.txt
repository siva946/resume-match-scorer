TESTING INSTRUCTIONS FOR MICROSOFT REVIEWERS

The extension works in TWO MODES:

1. DEMO MODE (No backend required - FOR TESTING)
   - Extension automatically detects if backend is unavailable
   - Shows demo match scores (45-95%) based on job keywords
   - All features are visible and testable
   - "Demo Mode" indicator appears on widget

2. FULL MODE (Backend required - FOR END USERS)
   - Requires Jobalytics backend on localhost:8000
   - Real AI-powered match scores using SentenceTransformers
   - Full job saving functionality

HOW TO TEST (NO SETUP REQUIRED):

1. Install the extension
2. Visit any of these sites:
   - https://www.naukri.com/software-engineer-jobs
   - https://www.indeed.com/jobs?q=software+engineer

3. Click on any job posting
4. Wait 3 seconds - match score widget appears automatically
